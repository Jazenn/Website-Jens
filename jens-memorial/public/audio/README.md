# Background music

Place the licensed ambient MP3 for the constellation page here:

```text
constellation-ambient.mp3
```

The app loads it from:

```text
/audio/constellation-ambient.mp3
```

Recommended export:

- MP3
- 30 to 60 minutes
- Stereo
- 128-192 kbps
- Soft fade-in/fade-out if possible

The website also applies a fade-in and crossfade-style restart near the end of the track.
